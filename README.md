# rxconnection
okhttp using rxjava for speed up your connection API

### Include Library
Add it in your root build.gradle at the end of repositories:

```
allprojects {
   repositories {
    maven { url "https://jitpack.io" }
   }
}
```
Add dependencies :

```
dependencies {
     compile 'com.github.andrehardian:rxconnection:2.1.16'
}

```
